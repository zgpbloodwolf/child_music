# 静态资源目录

内置曲库的音频与封面资源放在本目录,文件名需与 `src/data/songs.ts` 中的字段一一对应。

## 目录约定

| 子目录 | 用途 | 对应字段 | 示例 |
| --- | --- | --- | --- |
| `audio/` | 音频文件(.mp3 等) | `Song.src` | `/static/audio/s001.mp3` |
| `covers/` | 封面图(.jpg / .png) | `Song.cover` | `/static/covers/s001.jpg` |

> 歌词采用**内嵌**方式,直接写在 `src/data/songs.ts` 的 `Song.lyric` 字段(lrc 文本),无需单独放歌词文件,跨端最稳。

## 说明

- 封面缺失时,组件会自动用渐变色块 + 歌名首字占位,无图也能正常展示。
- 首次放入示例音频与封面后,即可在 H5 端听到声音、看到封面。
- 替换示例数据:编辑 `src/data/songs.ts`,把示例歌曲换成你的真实资源。
