# 页面设计规划

> 儿童启蒙音频 App(儿歌 / 古诗 / 三字经 / 故事)的页面与交互规划。
> 标注:✅ 已实现 · 🆕 建议新增(待确认) · ⏳ 后续阶段
>
> **版本:** v0.1(2026/06/29)

---

## 1. 设计原则

- **听觉为主,视觉辅助**:核心是「听」,界面服务于快速找到并播放内容。
- **儿童友好**:大图标、大字号、圆角、明亮温暖;重要操作(播放)按钮足够大。
- **操作极简**:点击即播;收藏、搜索等家长常用功能放在易达位置。
- **后台播放优先**:任何页面都能看到「正在播放」并一键回到播放页。

## 4. 页面详细设计

### 4.1 首页 home ✅

**线框:**
```
┌──────────────────────────────┐
│  🎵 启蒙音频        🔍 搜索   │  顶部:应用名 + 搜索入口
├──────────────────────────────┤
│  分类                         │
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐ │
│ │儿歌│ │古诗│ │三字│ │故事│ │  分类卡片(横向滚动)
│ │ ♬  │ │ ✎  │ │ 📖 │ │ 📚 │ │  → 点击进歌单详情
│ └────┘ └────┘ └────┘ └────┘ │
├──────────────────────────────┤
│  全部音频                     │
│ ┌──────────────────────────┐ │
│ │[封面] 两只老虎    经典儿歌│ │  音频项 → 点击播放
│ ├──────────────────────────┤ │
│ │[封面] 咏鹅        骆宾王  │ │
│ ├──────────────────────────┤ │
° │ ...                      │ │
├──────────────────────────────┤
│ [▩] 小星星  ▶播放  ⏭下一首  │  迷你播放条(有内容时)
├──────────────────────────────┤
│   首页            我的        │  tabBar
└──────────────────────────────┘
```

**控件与功能:**

| 控件 | 功能 |
| --- | --- |
| 搜索入口(顶部) | 点击 → 搜索页 |
| 分类卡片(4 个) | 点击 → 对应歌单详情页(`?id=children/poetry/...`) |
| 音频列表项 | 点击 → 以全部音频为队列,从该项播放 |
| 迷你播放条 | 显示当前播放;点击 → 播放页;按钮控制播放/下一首 |

**状态:** 内置必有数据,无空状态。

### 4.3 搜索 search ✅

**线框:**
```
┌──────────────────────────────┐
│ [🔍 搜索音频 / 作者 / 专辑  ] │  输入框(实时过滤)
├──────────────────────────────┤

│ ...                          │

```

**控件与功能:**

| 控件 | 功能 |
| --- | --- |
| 搜索输入框 | 实时按 名称 / 作者 / 专辑 过滤内置曲库 |
| 结果列表项 | 以当前结果为队列,从点击项播放 |

**状态:** 初始提示 / 无结果提示。⏳ 后续可加搜索历史、热门词。

### 4.4 播放页 player ✅(核心,全屏沉浸式)

**线框:**
```
┌──────────────────────────────┐
│ ↓                             │  自定义导航:下拉/返回关闭
│                               │
│         ┌──────────┐         │
│         │          │         │
│         │   封面   │         │  大封面(播放时旋转/呼吸 🆕)
│         │          │         │
│         └──────────┘         │
│                               │
│  两只老虎              ♡     │  名称 + 收藏(♥ 已收藏)
│  经典儿歌                     │  作者/来源
│                               │
│       两只老虎 两只老虎       │  歌词区(随播放滚动高亮)
│       跑得快 跑得快           │  故事类 → 「暂无歌词」
│                               │
│  00:06  ═══●═══════  00:15    │  进度条(拖动松手 seek)
│                               │
│   顺序    ⏮   ▶   ⏭          │  模式/上一首/播放暂停/下一首
└──────────────────────────────┘
```

**控件与功能:**

| 控件 | 功能 |
| --- | --- |
| 返回 | `navigateBack` 回来源页 |
| 大封面 | 展示;🆕 播放时可加旋转/呼吸动画 |
| 收藏按钮 ♥ | 切换收藏(本地存储) |
| 歌词区 | 解析 lrc,随 `currentTime` 高亮并滚动到当前句;无歌词显示占位 |
| 进度条 slider | 拖动时显示本地值,松手 seek 一次(避免频繁 seek) |
| 播放模式 | 顺序 / 单曲循环 / 随机,循环切换 |
| 上一首 / 播放暂停 / 下一首 | 基本播放控制 |

**状态:**
- 加载中:`onWaiting`/`onCanplay` 期间显示加载态(🆕 可加 loading 指示)
- 播放中 / 暂停:按钮文案/图标切换
- 无歌词:故事类显示「暂无歌词」
- 🆕 当前播放队列入口(查看整张列表)—— ⏳ 后续阶段

## 5. 状态汇总(全局)

| 状态 | 处理 |
| --- | --- |
| 正在播放 | 迷你播放条显示;播放页按钮为「暂停」 |
| 暂停 | 迷你播放条仍显示;播放页按钮为「播放」 |
| 加载/缓冲 | 🆕 播放页可显示 loading(当前仅控制台日志) |
| 播放出错 | `onError` → 暂停态;🆕 可加错误提示(当前仅控制台) |
| 空列表 | 各页均有空状态文案 |
| 未收藏 | 我的页引导文案 |

## 6. 待确认的设计决策

以下决策会影响后续视觉与开发,需要你拍板:

1. **视觉风格 / 配色**:当前主色为红 `#ff4d4f`。儿童启蒙向可考虑更温暖活泼的配色(如暖橙、草绿、天蓝)。是否调整主题色?
2. **tabBar 图标**:当前纯文字。是否准备图标资源(首页/我的 各一对)?
3. **播放页动效**:封面是否加播放旋转/呼吸动画?
4. **播放队列入口**:播放页是否加「当前队列」入口(⏳ 当前登记为后续阶段)?
5. **页面增减**:是否需要新增页面(如「分类」独立 tab、「最近播放」区、「音频详情」页)?
